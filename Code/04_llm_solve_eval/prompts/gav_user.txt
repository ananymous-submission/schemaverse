## Role
You are a data integration expert.

## Task
Design a unified global schema and express it as SQL **CREATE VIEW** statements.

When constructing the views, you may:
- Rename columns to produce consistent and meaningful attribute names.
- Use relational operations such as JOIN and UNION ALL where appropriate.
- Combine information across multiple local schemas when necessary to remove redundancy.

Reference local tables using `local_N.table_name` (e.g., `local_1.my_table`).

## Output Format
Output ONLY valid SQL CREATE VIEW statements separated by semicolons.
No explanations. No markdown. No CREATE TABLE statements.

Example:
CREATE VIEW example_view AS
SELECT 1 AS example_col
FROM local_1.example_table;
