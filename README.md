# SchemaVerse

SchemaVerse is a benchmark for global schema construction from fragmented local schemas.

## Description
SchemaVerse generates controlled schema integration scenarios by partitioning global schemas into multiple local schemas and applying structural and lexical transformations.

## Source Datasets
SchemaVerse is derived from publicly available datasets:
- BIRD
- Spider2
- Beaver
- ScienceBenchmark

We follow the licenses and terms of use of the original datasets.

## License

## Intended Use
This benchmark is intended for research in schema integration, data management, and evaluation of language models.
